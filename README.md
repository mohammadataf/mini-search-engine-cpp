# Mini Search Engine (C++)

A lightweight search engine built in C++ that processes text documents, performs text preprocessing, builds an inverted index, and supports efficient keyword-based document retrieval.

## Overview

This project demonstrates the core principles used by modern search engines, including document processing, tokenization, indexing, and search optimization. The goal is to understand how large-scale search systems organize and retrieve information efficiently.

## Features

### Current Features

* Read multiple text documents
* File handling using C++
* Document content extraction
* Character and document statistics

### Planned Features

* Text tokenization
* Case normalization
* Stop-word removal
* Inverted index construction
* Keyword-based search
* Multi-word query support
* Document ranking based on term frequency
* Phrase search
* Persistent index storage
* Interactive command-line interface
